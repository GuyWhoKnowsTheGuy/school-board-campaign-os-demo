# Scaling Roadmap: CCSD 46 to Statewide Adoption

## Phase 1: CCSD 46 reference implementation

- Build dashboard and docs.
- Validate with former candidate.
- Add public-source district intelligence.
- Create first tribal knowledge cards.
- Run a simulated candidate onboarding.

## Phase 2: Lake County district pack template

- Extract district schema.
- Add district discovery checklist.
- Build election calendar generator.
- Create source quality standards.
- Add import format for board members, schools, enrollment, finance, and election authority links.

## Phase 3: Illinois school board candidate OS

- Generalize Illinois election mechanics.
- Add Illinois-specific filing and campaign finance warnings.
- Add district-type variations.
- Build reusable candidate readiness and petition workflows.

## Phase 4: Multi-state architecture

- State-specific legal/election packs.
- District-specific intelligence packs.
- National core OS modules.
- Local stewardship networks.

## Phase 5: Ecosystem

- Candidate alumni knowledge commons.
- Public dashboard generator.
- Training curriculum.
- Partner civic organizations.
- Evidence-backed school governance knowledge base.
